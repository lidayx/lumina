# 编码解码功能测试用例

## 一、URL 编码/解码

### URL 编码

| 输入 | 预期输出 | 说明 |
|------|---------|------|
| `url encode hello world` | `hello world → hello%20world` | 英文空格编码 |
| `hello world url编码` | `hello world → hello%20world` | 中文关键词 |
| `url encode 你好世界` | `你好世界 → %E4%BD%A0%E5%A5%BD%E4%B8%96%E7%95%8C` | 中文编码 |
| `url encode test@example.com` | `test@example.com → test%40example.com` | 特殊字符编码 |
| `url encode a+b=c` | `a+b=c → a%2Bb%3Dc` | 多个特殊字符编码 |

### URL 解码

| 输入 | 预期输出 | 说明 |
|------|---------|------|
| `url decode hello%20world` | `hello%20world → hello world` | 英文空格解码 |
| `hello%20world url解码` | `hello%20world → hello world` | 中文关键词 |
| `url decode %E4%BD%A0%E5%A5%BD` | `%E4%BD%A0%E5%A5%BD → 你好` | 中文解码 |
| `url decode test%40example.com` | `test%40example.com → test@example.com` | 特殊字符解码 |

## 二、HTML 编码/解码

### HTML 编码

| 输入 | 预期输出 | 说明 |
|------|---------|------|
| `html encode <div>test</div>` | `<div>test</div> → &lt;div&gt;test&lt;/div&gt;` | HTML 标签编码 |
| `<div>test</div> html编码` | `<div>test</div> → &lt;div&gt;test&lt;/div&gt;` | 中文关键词 |
| `html encode a & b` | `a & b → a &amp; b` | 特殊字符 & 编码 |
| `html encode "hello"` | `"hello" → &quot;hello&quot;` | 引号编码 |
| `html encode <script>alert('xss')</script>` | `<script>alert('xss')</script> → &lt;script&gt;alert(&#39;xss&#39;)&lt;/script&gt;` | XSS 防护示例 |

### HTML 解码

| 输入 | 预期输出 | 说明 |
|------|---------|------|
| `html decode &lt;div&gt;test&lt;/div&gt;` | `&lt;div&gt;test&lt;/div&gt; → <div>test</div>` | HTML 标签解码 |
| `&lt;div&gt;test&lt;/div&gt; html解码` | `&lt;div&gt;test&lt;/div&gt; → <div>test</div>` | 中文关键词 |
| `html decode a &amp; b` | `a &amp; b → a & b` | 特殊字符 & 解码 |
| `html decode &quot;hello&quot;` | `&quot;hello&quot; → "hello"` | 引号解码 |

## 三、Base64 编码/解码

### Base64 编码

| 输入 | 预期输出 | 说明 |
|------|---------|------|
| `base64 encode hello world` | `hello world → aGVsbG8gd29ybGQ=` | 英文编码 |
| `hello world base64编码` | `hello world → aGVsbG8gd29ybGQ=` | 中文关键词 |
| `base64 encode 你好` | `你好 → 5L2g5aW9` | 中文编码 |
| `base64 encode test` | `test → dGVzdA==` | 短字符串编码 |

### Base64 解码

| 输入 | 预期输出 | 说明 |
|------|---------|------|
| `base64 decode aGVsbG8gd29ybGQ=` | `aGVsbG8gd29ybGQ= → hello world` | 英文解码 |
| `aGVsbG8gd29ybGQ= base64解码` | `aGVsbG8gd29ybGQ= → hello world` | 中文关键词 |
| `base64 decode 5L2g5aW9` | `5L2g5aW9 → 你好` | 中文解码 |
| `base64 decode dGVzdA==` | `dGVzdA== → test` | 短字符串解码 |

### Base64 错误处理

| 输入 | 预期输出 | 说明 |
|------|---------|------|
| `base64 decode invalid!` | 错误提示 | 无效的 Base64 字符串 |

## 五、使用示例

### 示例 1：URL 编码一个带参数的 URL
```
输入：url encode https://example.com?q=hello world&page=1
输出：https://example.com?q=hello world&page=1 → https%3A%2F%2Fexample.com%3Fq%3Dhello%20world%26page%3D1
```

### 示例 2：HTML 编码防止 XSS
```
输入：html encode <img src=x onerror=alert(1)>
输出：<img src=x onerror=alert(1)> → &lt;img src=x onerror=alert(1)&gt;
```

### 示例 3：Base64 编码二进制数据
```
输入：base64 encode {"name":"test","value":123}
输出：{"name":"test","value":123} → eyJuYW1lIjoidGVzdCIsInZhbHVlIjoxMjN9
```

### 示例 4：MD5 校验
```
输入：md5 mypassword
输出：mypassword → <32位MD5哈希值>
```

- ✅ HTML 解码：正确解码 HTML 实体
- ✅ Base64 解码：正确处理填充字符

- ✅ 无效的 URL 编码返回错误
- ✅ 所有错误都有清晰的错误提示

### 5. 中文支持
- ✅ 所有编码方式都支持中文
- ✅ 编码解码结果正确
